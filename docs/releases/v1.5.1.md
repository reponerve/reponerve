# RepoNerve v1.5.1

Release Date: 2026-06-24

Status: **Released**

---

# Summary

Patch release: CLI **version identification** — `--version` flag and `version` subcommand.

---

# Highlights

## `reponerve --version` / `reponerve version`

* Prints semver and build metadata (commit, build date) on release binaries
* Defaults to `dev` for local `go build` without ldflags
* GoReleaser and `make build` inject version via `internal/version`

---

# Upgrade

```bash
npm install -g reponerve@1.5.1
# or
curl -fsSL https://raw.githubusercontent.com/reponerve/reponerve/main/scripts/install.sh | bash

reponerve --version
```
