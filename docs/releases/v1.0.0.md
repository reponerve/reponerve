# Product Mission

Software Understanding is the primary outcome. RepoNerve preserves, organizes, and transfers software knowledge so that understanding survives beyond individual contributors.

```text
Knowledge Preservation → Software Memory → Software Understanding
```

# Major Capabilities

## Repository Foundations

* Repository initialization, configuration, SQLite storage, migrations
* Git + ADR ingestion via `reponerve scan`

## Repository Memory

* Event, decision, intent, and fact extraction
* Memory linking, query, and explanation flows

## Code Intelligence

* Go indexing via `go/ast`; 19 additional languages via Tree-sitter (`gotreesitter`)
* Symbols, imports, call graphs, deterministic entity IDs
* `explain-file`, symbol resolution, impact analysis

## Repository-Code Linking

* Deterministic links between repository memory and code entities
* Linker extracts references from ADRs, commits, and docs

## Development Experience

CLI: `ask`, `explain`, `explain-file`, `explain-function`, `explain-struct`, `explain-interface`, `explain-type`, `plan`, `impact`, `review`, `onboard`

Token-efficient output: `--format compact`, `--token-budget`

## Token Intelligence (ISSUE-060)

* Graph-aware context compression
* MCP/CLI format parity
* `reponerve hook install` for incremental scan

## Evidence Graph & Session Memory (ISSUE-061)

* Community detection, `reponerve explore`, graph MCP tools
* `remember` / `forget`, handoff bundles, workflow templates

## MCP Integration

* 43 tools — memory, context, ownership, graph, development experience, session memory

## Agent Intelligence

* Context packages, search, session memory, workflow templates

# Known Limitations

* Deterministic, evidence-backed retrieval only (no semantic embeddings)
* Fixed workflow templates (no user-defined composition)
* Local-first (no mandatory cloud)
* Single-repository depth (no cross-repo federation)

See `docs/roadmap/v1.x-backlog.md` for post-v1.0 RFC items.

# Upgrade

```bash
go install github.com/reponerve/reponerve/cmd/reponerve@latest
reponerve init
reponerve scan
reponerve hook install   # optional: refresh after commits
```

Cursor / Claude: configure MCP per `docs/ai-chat-integration.md`.
