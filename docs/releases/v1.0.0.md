# RepoNerve v1.0.0

Release Date: 2026-06-08

Status: Release Ready

---

# Project Overview

RepoNerve is a local-first repository intelligence platform for humans and AI agents.

It transforms repository artifacts into structured repository memory, derives explainable repository intelligence, and exposes that intelligence through agent-oriented packaging and MCP.

---

# Architecture Summary

RepoNerve v1.0 preserves a layered architecture:

Repository
↓
Ingestion
↓
Memory
↓
Context
↓
Ownership
↓
Knowledge Graph
↓
Repository Intelligence
↓
Agent Intelligence
↓
MCP
↓
AI Agents

The v1.0 release keeps repository intelligence authoritative and treats agent-facing layers as packaging and orchestration layers rather than new sources of truth.

# Known Limitations

* Search remains deterministic and evidence-based only.
* Semantic search, hybrid search, and search adapters are intentionally excluded from v1.0.0.
* Sessions and workflows are in-memory packaging artifacts and are not persisted.
* RepoNerve remains local-first and does not introduce hosted infrastructure.
* Benchmark-grade performance coverage is not yet part of the automated release gate.
