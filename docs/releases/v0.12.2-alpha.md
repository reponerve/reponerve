# RepoNerve v0.12.2-alpha

Release Date: 2026-06-16

Status: Released (engineering iteration — chat without MCP)

Milestone: AI chat without MCP

---

## Summary

v0.12.2-alpha lets agents use RepoNerve in LLM chat **without MCP** — the same evidence contract via CLI `--json`, skill workflow, and `CLAUDE.md` hooks.

---

## Features

### Chat without MCP

- `--json` on all Development Experience CLI commands — emits MCP-compatible envelope (`structured`, `agent`, `formatted`)
- Skill trigger `/reponerve ask "..."` with mandatory CLI workflow
- `CLAUDE.md` RepoNerve section installed by `reponerve init` / `integrate`
- **`ask` only** — no separate `query` command

### Integration

- Updated skill, rule, bundle, and `docs/ai-chat-integration.md`
- Removed third-party product references from docs

---

## Upgrade

```bash
go install ./cmd/reponerve
reponerve integrate --force
```

Restart MCP in IDE if used; chat-without-MCP works via `reponerve ask "..." --json`.
