# Release v0.15.0-alpha — Multi-Language Code Intelligence (ISSUE-062)

Date: 2026-06-11

Milestone: ISSUE-062 Multi-Language Code Intelligence

---

## Summary

Closes ISSUE-062 acceptance criteria: Tree-sitter indexing for TypeScript, Python, and Rust using the same code intelligence storage model as Go. Final v0.x engineering checkpoint before v1.0.0 release review.

---

## Features

### Tree-sitter parsers (pure Go, no CGO)

- `internal/code/lang/` — deterministic symbol and import extraction via `gotreesitter`
- TypeScript (`.ts`, `.tsx`): functions, classes, interfaces, imports
- JavaScript (`.js`, `.jsx`, `.mjs`, `.cjs`): functions, classes, imports
- Python (`.py`): functions, classes, imports
- Rust (`.rs`): functions, structs, traits, impl methods, `use` imports
- Java (`.java`): classes, interfaces, methods, imports
- C# (`.cs`): classes, interfaces, methods, `using` directives
- Ruby (`.rb`): classes, modules, methods, `require` imports
- Kotlin (`.kt`, `.kts`): classes, interfaces, functions, imports
- Swift (`.swift`): classes, protocols, functions, imports
- PHP (`.php`): classes, interfaces, methods, namespace/use imports
- C (`.c`, `.h`): structs, functions, `#include` imports
- C++ (`.cpp`, `.cc`, `.hpp`, …): classes, methods, functions, includes
- Scala (`.scala`): classes, traits, objects, methods, imports
- Lua (`.lua`): module tables, functions, methods, `require` imports
- Bash (`.sh`, `.bash`): functions, `source` imports
- SQL (`.sql`): tables and views
- Dart (`.dart`): classes, methods, functions, imports
- Elixir (`.ex`, `.exs`): modules, functions, imports
- Zig (`.zig`): structs, methods, functions, `@import`

### Indexing

- `reponerve scan` indexes mixed repos (Go + TS + Python + Rust)
- Module discovery via `package.json`, `pyproject.toml`, `Cargo.toml`
- Incremental re-index respects all supported source files and project markers

### Development Experience

- `explain-file` and symbol resolution work for indexed non-Go files
- Repository-code linking unchanged — works across all indexed languages
