# RepoNerve v0.12.1-alpha

Release Date: 2026-06-16

Status: Released (engineering iteration — AI integration)

Milestone: Universal AI chat + automatic IDE setup

---

## Summary

v0.12.1-alpha makes RepoNerve work as a first-class AI integration across IDEs and LLMs — not MCP-only. `reponerve init` now installs Cursor skill, MCP configs, and related IDE files automatically.

---

## Features

### Automatic IDE integration

- `reponerve init` installs project integration:
  - `.cursor/mcp.json`, `.cursor/skills/reponerve/`, `.cursor/rules/reponerve.mdc`
  - `.vscode/mcp.json`, `.continue/mcpServers/reponerve.json`
- Global Cursor skill at `~/.cursor/skills/reponerve/`
- `reponerve integrate` refreshes configs; merges MCP without removing other servers
- `reponerve init --skip-ide` to opt out

### Universal AI chat

- `docs/ai-chat-integration.md` — direct chat in any IDE/LLM
- Updated compatibility matrix (38 MCP tools, major IDEs)
- Skill documents MCP + CLI fallback (works without MCP)

### Implementation

- `internal/integration/` — embedded bundle + install logic
- `internal/cli/integrate/` — standalone integrate command

---

## Upgrade

```bash
go install ./cmd/reponerve
reponerve integrate          # existing repos
# or
reponerve init && reponerve scan   # new repos
```

Restart MCP in your IDE after upgrading the binary.
