# RepoNerve v1.5.0

Release Date: 2026-06-24

Status: **Released**

---

# Summary

Minor release: **Local Explore UI** (RFC-009) — GitNexus-class local graph browser without a separate SPA.

---

# Highlights

## `reponerve explore --serve`

* Local UI on `127.0.0.1:8765` (default)
* **templ** + **htmx** + **Alpine.js** + **Cytoscape.js** (CDN)
* Filterable node table, evidence panel with edge JSON and MCP hints
* Capped at 200 nodes for responsive layout; full counts in header

## Static export

* `reponerve explore -o reponerve-graph.html` uses the same templ layout
* Interactive htmx requires `--serve`; static export includes cytoscape subgraph

---

# Upgrade

```bash
npm install -g reponerve@1.5.0
# or
curl -fsSL https://raw.githubusercontent.com/reponerve/reponerve/main/scripts/install.sh | bash

reponerve explore --serve
```
