# Release v0.14.0-alpha — Evidence Graph & Session Memory (ISSUE-061)

Date: 2026-06-11

Milestone: ISSUE-061 Evidence Graph & Session Memory

## Summary

Closes ISSUE-061 acceptance criteria: graph communities and discovery, token-budget traversal, `reponerve explore`, session memory CLI/MCP, agent handoff bundles, and workflow templates.

---

## Features

### Evidence Graph Intelligence

- `internal/graph/communities/` — connected-component detection with deterministic ordering
- `internal/graph/discovery/` — god nodes, cross-type surprises, suggested questions
- `internal/graph/traversal/budget.go` — token-budget BFS via `TraverseWithBudget`
- `reponerve explore` — HTML graph export with embedded JSON
- MCP tools: `discover_surprises`, `suggest_questions`, `query_graph`

### Session Memory

- `reponerve remember` / `reponerve forget` — CLI + MCP (`remember`, `forget`)
- Session writeback — Q&A creates traceable facts with `session` provenance
- Temporal relevance — access-aware ranking via `.reponerve/session-access.json`
- `reponerve handoff export|import` — deterministic context transfer between sessions
- `reponerve workflow onboarding|review|change` — fixed workflow presets

## Upgrade

```bash
go install github.com/reponerve/reponerve/cmd/reponerve@latest
reponerve explore --output graph.html   # visualize knowledge graph
reponerve remember "SQLite is local-first"   # persist session fact
```

MCP graph discovery example:

```json
{ "name": "discover_surprises", "arguments": { "repository_id": "..." } }
```
