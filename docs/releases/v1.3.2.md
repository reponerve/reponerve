# RepoNerve v1.3.2

Release Date: 2026-06-24

Status: **Released**

---

# Summary

Patch release: **npm distribution** (RFC-006) for Node.js/TypeScript teams.

---

# Highlights

## npm install

* Package `reponerve` at `npm/reponerve/`
* `postinstall` downloads the same GoReleaser binary as GitHub Releases
* `npm install -g reponerve` or `npm install -D reponerve` + `npx reponerve`
* CI workflow `.github/workflows/npm-publish.yml` publishes on tag when `NPM_TOKEN` is set

## Documentation

* `docs/rfc/RFC-006-npm-distribution.md`
* `docs/install.md` — npm section
* README — npm install path

# Upgrade

```bash
npm install -g reponerve@1.3.2
```

Or use the install script (unchanged):

```bash
curl -fsSL https://raw.githubusercontent.com/reponerve/reponerve/main/scripts/install.sh | bash
```
