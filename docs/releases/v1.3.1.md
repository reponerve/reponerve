# RepoNerve v1.3.1

Release Date: 2026-06-24

Status: **Released**

---

# Summary

Patch release: **binary-first installation** for non-Go users, plus GoReleaser-published archives for all platforms.

---

# Highlights

## Installation (no Go required)

* `scripts/install.sh` — one-liner from GitHub Releases
* `docs/install.md` — canonical install guide (manual download, Windows, version pins)
* README and `docs/ai-chat-integration.md` lead with the install script
* `make install` for Go developers
* PR workflow template uses install script instead of `go install`

## Release process

* GoReleaser publishes platform archives on tag push (do not create empty releases with `gh release create` alone)

---

# Upgrade

```bash
curl -fsSL https://raw.githubusercontent.com/reponerve/reponerve/main/scripts/install.sh | bash
```

Or download `reponerve_1.3.1_<os>_<arch>.tar.gz` / `.zip` from GitHub Releases.
