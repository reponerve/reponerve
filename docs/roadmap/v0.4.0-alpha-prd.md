# RepoNerve v0.4.0-alpha PRD

## Objective

Transform repository memory into AI-ready context.

---

# Problem

The Memory Engine stores repository knowledge.

The Query Engine retrieves repository knowledge.

AI agents still need multiple commands to understand a repository.

---

# Solution

Introduce a Context Engine.

The Context Engine assembles repository knowledge into a concise repository briefing suitable for AI coding agents.

---

# User Story

As an AI coding agent,

I want a repository context summary,

So that I can understand the repository without scanning every file.
