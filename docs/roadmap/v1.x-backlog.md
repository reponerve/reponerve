# RepoNerve Out-of-Scope Items

Version: 1.1

Status: Draft

Updated: 2026-06-11

---

# Purpose

RepoNerve has **one product release: v1.0.0**.

All product capabilities — including token intelligence, graph discovery, session memory, and multi-language support — are delivered through **v0.x.0-alpha iterations** and ship in v1.0. See `docs/roadmap/v1.0-iteration-plan.md`.

This document lists capabilities **agreed out of v1.0 scope**. They are not in the v0.10–v1.0 iteration plan and are not planned for a separate v1.x product release. Reconsideration requires a post-v1.0 RFC.

---

# Out of Scope for v1.0

## Semantic Search

Embedding-based or vector retrieval as a primary authority.

## Hybrid Search

Combined deterministic and semantic retrieval adapters where vectors override evidence.

## User Defined Workflows

Custom workflow composition beyond the fixed v1.0 workflow template set (ISSUE-061).

## Autonomous Code Modification

RepoNerve does not autonomously write, commit, or deploy code.

## Cloud-Required Core Product

Core v1.0 must run local-first without mandatory SaaS.

## Cross-Repo Enterprise Federation

Org-wide multi-repository federation at Sourcegraph scale.

## Visual Graph UI (Full Product)

Dedicated web/TUI graph explorer beyond `reponerve explore` HTML export (ISSUE-061).

---

# Rule

If a capability is in `docs/roadmap/v1.0-iteration-plan.md` or `docs/vision/vision.md` v1.0 scope, it belongs to **v1.0**.

If a capability is listed in this document, it is excluded from v1.0.

There is **no v1.x product roadmap**. After v1.0.0 ships, future work requires a new RFC — not an implicit v1.1 track.
