# RepoNerve v0.5.0-alpha PRD

## Objective

Expose repository intelligence to AI coding agents through MCP.

---

# Problem

RepoNerve already understands:

* Repository history
* Decisions
* Intents
* Facts
* Relationships
* Repository context

However, AI agents cannot consume this knowledge directly.

---

# Solution

Introduce an MCP Server.

The MCP Server exposes RepoNerve capabilities through the Model Context Protocol.

---

# User Story

As an AI coding agent,

I want access to repository memory and context,

So that I can understand a repository without scanning its entire codebase.
