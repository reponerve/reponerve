# RepoNerve v0.6.0-alpha PRD

## Objective

Enable AI coding agents to reason over repository intelligence.

---

# Problem

RepoNerve can:

* Extract repository knowledge
* Query repository knowledge
* Generate repository context
* Expose knowledge through MCP

However, agents still need to manually assemble knowledge into useful reasoning.

---

# Solution

Introduce Agent Intelligence.

Agent Intelligence transforms repository knowledge into actionable insights for AI coding agents.

---

# User Story

As an AI coding agent,

I want repository insights and reasoning assistance,

So that I can understand a repository faster and make better changes.

---

## Impact Analysis
