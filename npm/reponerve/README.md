# reponerve (npm)

npm distribution for the [RepoNerve](https://github.com/reponerve/reponerve) CLI.

Installs the prebuilt native binary from GitHub Releases on `postinstall`. Node.js is only required for installation — runtime is the Go binary.

```bash
npm install -g reponerve
# or per project
npm install -D reponerve
npx reponerve init
```

See `docs/install.md` in the repository root.
