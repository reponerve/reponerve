# RepoNerve

RepoNerve is an open-source memory and context engine for software repositories.

Software remembers code.

Software forgets context.

RepoNerve preserves repository knowledge, decisions, intent, ownership, patterns, and history so that both humans and AI systems can understand software without repeatedly rediscovering the same information.

## Core Goals

- Preserve engineering knowledge
- Explain software evolution
- Improve repository understanding
- Generate optimized context for development
- Reduce unnecessary AI token consumption
- Enable AI agents to work with repository-specific memory

## Philosophy

Memory first.

Context second.

AI third.
