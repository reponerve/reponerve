# Changelog

All notable changes to RepoNerve will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-08

### Added
- Agent Context Builder for AI-driven repository understanding
- Repository Search with semantic and hybrid capabilities
- Agent Session Intelligence for workflow context preservation
- Workflow Intelligence for multi-step task coordination
- Complete MCP Server implementation with all core capabilities
- Knowledge Graph with relationship traversal and impact analysis
- Ownership Intelligence for contributor expertise detection
- Repository Intelligence for unified context generation
- Memory Engine with deterministic fact, intent, decision, and event extraction
- Query Engine with trace, list, get, and explain operations
- Context Engine with generation and export workflows
- Ingestion Engine for repository scanning and artifact extraction
- Full CLI with comprehensive command structure

### Changed
- Improved documentation and architecture specifications
- Enhanced context generation with compression capabilities
- Optimized memory storage with deterministic ordering
- Streamlined agent integration patterns

### Fixed
- Memory linking determinism across all operations
- Context generation accuracy and consistency
- Graph traversal performance and correctness
